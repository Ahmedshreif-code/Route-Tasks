## 🚀 Download a Specific Task

If you only need **one task folder** (instead of downloading the entire repository), you can use:

**https://download-directory.github.io/**

### Steps
1. Open this repository on GitHub.
2. Navigate to the folder of the task you want.
3. Copy the URL of that folder.
4. Open **https://download-directory.github.io/**
5. Paste the folder URL and click **Enter**.
6. The selected folder will be downloaded as a ZIP file.
